# Scope Statement

## Project Scope

The University Course Registration System will provide a centralized platform for managing academic course registration activities.

## In Scope

### Student Features

* User login and authentication
* View available courses
* Register for courses
* Drop courses
* View timetable
* Track registration status

### Academic Advisor Features

* Review registrations
* Approve or reject course selections

### Lecturer Features

* View enrolled students

### Administrator Features

* Manage courses
* Manage semesters
* Generate reports

## Out of Scope

* Learning Management System (LMS) integration
* Tuition fee payment processing
* Student grading functionality
* Attendance management
* Library management services

## Deliverables

* Requirements Documentation
* Process Models
* UML Diagrams
* Risk Register
* Product Backlog
* Final Case Study Report

## Project Boundaries

The project focuses solely on course registration activities and does not cover broader university management functions.
