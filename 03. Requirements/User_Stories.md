# User Stories

## Student Stories

### US-01

As a student, I want to view available courses so that I can choose subjects for the upcoming semester.

### US-02

As a student, I want to register for courses online so that I can complete enrollment without visiting the registrar office.

### US-03

As a student, I want to view my timetable so that I can manage my academic schedule effectively.

### US-04

As a student, I want to receive registration status updates so that I know whether my course selections have been approved.

## Academic Advisor Stories

### US-05

As an academic advisor, I want to review student registrations so that I can ensure compliance with academic requirements.

### US-06

As an academic advisor, I want to approve or reject registrations so that students follow their academic pathway correctly.

## Lecturer Stories

### US-07

As a lecturer, I want to access class lists so that I can prepare course materials and attendance records.

## Administrator Stories

### US-08

As an administrator, I want to manage course offerings so that students have access to accurate course information.

### US-09

As an administrator, I want to generate reports so that I can monitor enrollment trends and registration activity.
